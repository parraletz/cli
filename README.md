kueski cli

Usege:

## With parameters:

```
./cli.py salve vera mundi
```

## With flags:


```
./cli.py -x -y
```

## With key and value

```
./cli.py -k hola=mundo saludos=terricolas
```

## With parameters, flags and key/value

````
./cli.py hola mundo salve vera mundi -x -y --kwargs hola=mundo saludos=terricolas
```
